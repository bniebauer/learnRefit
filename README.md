# Learn Refit

## Description
This is a sample project using Refit, a .NET package for setting up quick web API calls.
In this example I am using the GitHub web API to get public User data and Repo data.

## Break Down
This solution is made up of three projects:

* A Class Library to create the Refit Service
* A Unit Test to test the Class Library
* A Console Application to use the Class Library
